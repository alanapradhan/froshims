YageZine Theme

Readme file of YageZine Theme

  1. Upload "YageZine" folder to themes directory (/wp-content/themes/)

  2. Install this theme

    Click on image and click "Activate YageZine"

  3. After Install this theme, go to "New Casino Mag Options" (Appearance -> YageZine Options)

  4. And than, you can see the New Casino Mag Options

  5. This is preview for Featured setting

  6. This is preview for Sliding Panel

  7. If you want add image on "sliding post" box and "featured post", don't forget add "Costum Fields" when you create post "thumbnail" for sliding image and "featured" for featured post.

    For best resolution (image) use "205px * 160px" on sliding post and "304px * 190px" on featured post
